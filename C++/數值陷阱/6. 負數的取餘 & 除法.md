6. 負數的取餘 & 除法
觀念講解

C++ 中 -5 / 2 == -2、-5 % 2 == -1：商向零捨入，餘數可為負。

對數學取餘（floor modulus）需特別處理。

cpp
複製
編輯
#include <iostream>
using namespace std;

int main() {
    cout << "-5 / 2 = " << (-5 / 2) << endl;  // -2
    cout << "-5 % 2 = " << (-5 % 2) << endl;  // -1

    // 如果要 floor division：
    int a = -5, b = 2;
    int q = (a - (a % b)) / b;    // -3
    int r = a - q * b;            // 1
    cout << "floor div: q=" << q << ", r=" << r << endl;
    return 0;
}
