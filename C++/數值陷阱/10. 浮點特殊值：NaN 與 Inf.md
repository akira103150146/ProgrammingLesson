10. 浮點特殊值：NaN 與 Inf
觀念講解

IEEE 754 浮點支援 +∞、-∞、NaN。

標準函式庫提供 std::isnan()、std::isinf() 做判斷。

cpp
複製
編輯
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    double inf = 1.0 / 0.0;      // +Inf
    double nan = 0.0 / 0.0;      // NaN

    cout << "isinf(inf)? " << isinf(inf) << endl;
    cout << "isnan(nan)? " << isnan(nan) << endl;
    return 0;
}
